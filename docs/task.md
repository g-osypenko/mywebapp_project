# Індивідуальне завдання (Варіант 20)

## Розрахунок параметрів

Номер у списку (або залікової книжки):  **N = 20** .

* **Спосіб конфігурації та БД:** $V_2 = (20 \% 2) + 1 = 1$. Конфігурація через аргументи командного рядка. База даних —  **MariaDB** .
* **Тематика застосунку:** $V_3 = (20 \% 3) + 1 = 3$. **Simple Inventory** (сервіс обліку обладнання).
* **Порт застосунку:** $V_5 = (20 \% 5) + 1 = 1$. Застосунок слухає порт  **8080** .

## Опис предметної області (Simple Inventory)

Об'єкт інвентарю містить наступні поля:

* `id` (int, Primary Key)
* `name` (string)
* `quantity` (int)
* `created_at` (datetime)
