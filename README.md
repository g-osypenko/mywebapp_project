# Лабораторна робота №1: Розгортання Web-сервісу

Цей репозиторій містить виконання лабораторної роботи №1. Проєкт являє собою сервіс "Simple Inventory" (облік обладнання), написаний на Python з використанням фреймворку FastAPI та бази даних MariaDB.

Застосунок розгортається автоматично на віртуальній машині (Ubuntu Server) за допомогою bash-скрипта, ізолюється через Systemd та працює за Reverse Proxy (Nginx) з обмеженнями UFW.

## Структура документації

Документація розділена на логічні частини. Будь ласка, перейдіть за посиланнями нижче для отримання деталей:

1. [Варіант індивідуального завдання (Розрахунки)](docs/task.md "null") — опис варіанту 20 та його параметрів.
2. [Документація API](docs/api.md "null") — перелік ендпоінтів, їх призначення та правила Content Negotiation.
3. [Інструкція з розгортання та тестування](docs/deployment.md "null") — детальний гайд по підготовці віртуальної машини, облікових записах, мережевих обмеженнях та перевірці працездатності.

## Швидкий запуск

Для розгортання системи скопіюйте репозиторій на свіжу віртуальну машину (Ubuntu Server 22.04) і виконайте скрипт розгортання з правами root:

```
sudo bash scripts/deploy.sh
```

Після завершення веб-застосунок буде доступний за IP-адресою вашої віртуальної машини на стандартному порту 80.
